VÍ DỤ IF

Từ khóa if dùng để bắt đầu một cấu trúc lệnh có điều kiện thực thi trong javascript, có cú pháp sau:
	if (điều kiện) câu lệnh;
	if (điều kiện) câu lệnh; else câu lệnh;

	var a = 5, b = 6, c = (a * b) / 2;
	if ( a > c )
		alert (“A lớn hơn C”);
	else
		alert(“C lớn hơnA”);
